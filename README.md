  <h2>简介<h2>"清泽心雨"网站是太原理工大学学生工作的门户网站。
"清泽"二字蕴涵“清兮汾水，惠泽三晋”之意，又可引申为知识的河流滋润着莘莘学子，与唐代诗人杜甫的佳句中“好雨知时节”、“润物细无声”所象征的“心雨滋润”相映成趣。旖旎葱郁的清泽园与温婉煌照的“清泽心雨”，在校内与网上之间，让点滴美妙不觉淌入心间。
清泽心雨 QQ：1448444888百度搜索：“清泽心雨”，第一个即为官网链接。</P>
<h3>发展历程<h3>
<ol>
<li>  2009年11月，清泽心雨上线，网站成立</li>
<li>  2009年12月，经过验收，被山西省教育厅评为“山西省高校思想政治教育主题网站达标单位”</li>
<li>  2010年，清泽心雨网站开始吸纳本科生，逐步构建由学生自主经营的学生网站</li>
<li>  2012年10月，清泽心雨周年专题上线，至此共计策划开发特色专题28个</li>
<li>  2012年11月，累计点击量突破四百万大关</li>
<li>  2012年12月，荣膺山西唯一第五届“全国高校百佳网站”称号</li>
<li>  2012年12月，获“全国大学生公益广告征集活动优秀组织奖</li>
<li>  2012年12月，荣获“太原理工大学2012年度宣传工作先进集体”</li>
<li>  2013年3月，荣获“太原理工大学第二届网站评比活动标兵网站”</li>
<li>  2013年，全国好声音优秀组织奖</li>
<li>  2013年5月，山西省省长李小鹏莅临我校指导期间，与我站数名学生进行交流，畅谈中国梦</li>
<li>  2013年5月，荣获“山西省青年五四奖状”</li>
<li>  2013年5月，获“全国大学生道德模范优秀组织学校”称号</li>
<li>  2013年5月，人民网山西频道视频创作基地</li>
<li>  2013年7月-8月，清泽心雨网站与人民网―山西频道合作，社会实践团队完成“梦想在路上”系列访谈节目</li>
<li>  2013年8月，荣膺第六届“全国高校百佳网站”称号</li> 
<li>  2013年12月，全国高校网站联盟常务理事单位</li>
<li>  2013年12月，第七届“全国高校百佳网站”荣誉称号</li>
<li>  2014年3月9日，“清泽心雨”网站第五届第三次全站大会；人民网―山西频道在我站建立“视频制作基地”，举行授牌仪式；明向校区常务理事会正式成立</li>
<li>  2014年3月31日，全国大学生公益广告征集活动校园巡讲太原理工大学站；新版清泽微视网站和清泽微视APP正式上线</li>
<li>  2014年08月14日，《人民日报》整版报道“清泽心雨”网站，称：“清泽心雨”已成为全国知名的网络文化育人引擎</li>
<li>  2015年1月，全国高校网站联盟网络资源建设委员会副主任单位</li>
<li>  2015年5月，全程记录报道“ASC15世界大学生超级计算机竞赛总决赛”</li>
<li>  2015年11月，联合山西省电视台举办“我型我秀”大型招聘类电视节目，并与同期参与太原理工大学宣传片的拍摄</li>
<li>  2016年4月，清泽心雨网站获"全国高校优秀网站"(总计二十个),清泽微视获"全国高校优秀网络栏目" (总计三十个) ，都是最高奖。</li>
<li>  2016年8月，全国百强最受欢迎校园媒体。</li>
<li>  2016年8月，中国软件产教互动联盟理事单位。</li>
<li>  2017年4月，清泽心雨荣获第三届中国青年志愿服务项目大赛金奖。</li>
<li>  2017年4月，全程记录报道第六届全国高校辅导员职业能力大赛。</li>  
<li>  2017年5月3日 山西省委书记、省人大常委会主任骆惠宁来到太原理工大学就加强和改进高校思想政治工作进行调研。期间，骆惠宁亲自参观了清泽心雨网络教育阵地并观看网站平台了解线上互动和交流情况，并且深入地与学生进行了交谈。</li>
<li>  2017年7月11日上午，教育部部长、党组书记陈宝生在山西省副省长张复明、省教育厅厅长吴俊清、校党委书记吴玉程和校长黄庆学的陪同下，来到位于我校明向校区学生活动中心二层的清泽心雨新媒体中心进行检查指导。陈宝生部长在新媒体中心听取了学生处处长郝根彦对清泽心雨新媒体中心发展情况的介绍，并与同学们进行了亲切交流</li>
</ol>
<h4>接下来介绍各个部门<h4>
<p>网络安全部</p>
简介：
网络安全部是清泽心雨网站负责服务器运维和软件开发的部门，保证清泽心雨下属的所有服务器、后台系统、微信后端等正常运行。以最专业的要求要求自己，能够完成复杂的业务逻辑软件开发和自动化服务器运维。下属的唯一子站蓝之青，是校内少有的具有校级组织资历认可的移动互联开发团队。
职能：
1、	负责清泽心雨主页服务器、清泽微视流媒体服务器、微信服务器等维护工作；
2、	负责移动端应用开发；
3、	负责面向企业的网站开发；
4、	负责代码审查，反逆向和防注入的安全性分析。

<p>视频集团</p>
<p>(1)清泽电视台</p>
    宗旨
做大学生自己的视频网站
<p>简介</P>
   清泽电视台作为清泽微视综合性最强的一个子站，拥有完备的影视设备，技术面包含摄影摄像、图像处理、特效制作和影视后期。清泽电视台在各方面都积极向大型电视台靠拢，承担着校园新闻、校园栏目等工作，鼓励学生把自己的创意通过影视的手段呈现在大家面前。秉承立足校园，服务师生的原则，引领正确的舆论导向，创造良好的舆论环境，紧紧围绕学校的中心工作和工作重心，及时宣传学校工作亮点，促进校园文化和精神文明建设。

    发展历程<ol>
<li>   2012年清泽微视上线 </li>
<li>  2013年电视台成立</li>
<li>   2015年改组，网络电视台成立</li>
<li>   2016年2月，改称清泽电视台</li>
<li>   2017年，二期清泽电视台传承并发展</li>
</ol>
<p>特色栏目</P>
   一周新闻播报：一档综合性视频新闻播报类节目。节目内容以重要时政、各项会议、校内各类活动、主流性报道等符合播出要求的新闻为主，传播校园先进文化，为师生了解校内重大新闻提供平台。 
   微伏私访：通过趣味性和实效性的采访反映学生对热点问题的看法和见解。以做大学生的生活代言者为理念，紧密围绕焦点话题和娱乐热点，反映当代大学生的精神风貌。
   太理杂货铺：杂货铺作为走在创意前沿的栏目，工作区间极为宽泛，主要以拍摄制作有创意的短片为主，内容以及形式都没有固化模式，为喜欢创作的小伙伴提供了广泛施展才华的空间！

<p>微影像</P>
宗旨：
做大学生自己的视频网站。

简介
<span>微影像</span>作为清泽微视的三个子站之一，旨在培养更多在视频制作方向具有较高水平的技术人才，制作更多我们大学生自己的视频。有时我们会置身活动现场，体验大学生的激情，也会进入生活，了解不同同学的不同体会。我们用镜头去体味生活，记录生活的点点滴滴。

组织任务及目标
承担学校的部分视频媒体宣传任务
承担微电影的创作任务

团队建设和发展
    微影像作为微视的子站，仅两年就从刚开始的校园新闻到现在独立拍摄制作微电影。在2016年，微影像开设了延时摄影、公益广告和恐怖题材微电影三个栏目，取得了一定的成功并且得到一定的关注和好评。这里有《微光音乐节》时我们的见证，有《你的吉他》的小清新与浪漫，有《莫名其妙的小偷》的慌张与惊吓。
